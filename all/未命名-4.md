```mermaid
开始
├─ 进入销售管理菜单
│  ├─ 选择操作类型
│  │  ├─ 1. 新建销售订单
│  │  │  ├─ 输入会员ID（可选，非会员跳过）
│  │  │  ├─ 验证会员有效性（若输入）
│  │  │  ├─ 添加图书商品
│  │  │  │  ├─ 输入图书ID和购买数量
│  │  │  │  ├─ 检查库存是否充足
│  │  │  │  │  ├─ 库存不足：提示"库存不足，无法销售"，返回添加商品步骤
│  │  │  │  │  └─ 库存充足：计算单价×数量=小计，加入订单明细
│  │  │  │  └─ 是否继续添加商品？（是/否）
│  │  │  ├─ 确认订单信息（商品列表、总金额）
│  │  │  ├─ 选择支付方式（现金/微信/支付宝等）
│  │  │  ├─ 生成订单编号，保存订单主表和明细
│  │  │  ├─ 扣减对应图书库存（调用库存模块接口）
│  │  │  ├─ 会员积分更新（若为会员，按消费金额累加积分）
│  │  │  └─ 显示订单成功信息（订单号、总金额）
│  │  ├─ 2. 查看销售记录
│  │  │  ├─ 选择查询方式（全部记录/按日期范围/按会员）
│  │  │  │  ├─ 全部记录：展示所有订单列表（订单号、日期、金额等）
│  │  │  │  ├─ 按日期范围：输入开始/结束日期，展示符合条件的订单
│  │  │  │  └─ 按会员：输入会员ID，展示该会员的所有订单及累计消费
│  │  │  └─ 选择订单查看详情（展示商品明细、支付方式等）
│  │  ├─ 3. 销售统计
│  │  │  ├─ 展示总销售额、今日销售额、总订单数
│  │  │  └─ （可选）按时间段/图书类别统计销售TOP榜
│  │  └─ 4. 返回主菜单
└─ 结束
```
